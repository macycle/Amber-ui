##Amber UI组件库

## 作者：macycle

## 简介
Amber UI 是一款基于Vue开发的简易版UI组件库。
该项目是我在深入了解Vue时做的一个项目，通过该项目，了解到了vue的很多属性。
该项目目前已经完成了Button、Icon、Grid、Popover、Toast、Input等简单组件。并且每个组件都完成了单元测试，保证了其可用性。
由于该项目是我个人作品，并没有经过多数人的检验，请勿用于正式项目。


## 安装

```
yarn add macycle-1
npm install macycle-1
```
